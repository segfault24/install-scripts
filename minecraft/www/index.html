Modded Client Installation

1) Install and run vanilla Minecraft at least once.
2) Install forge for Windows (choose Install Client).
3) Download and extract the mod pack.
4) Copy the extracted jar files to %appdata%/.minecraft/mods/ (typically C:\Users\<username>\AppData\Roaming\.minecraft\mods).
5) Start Minecraft and select the new Forge profile. It may take a while to load...

6) Optional: If you want a client-side mod that overlays info about what you're looking at, download the Waila mod into the same folder as above.

note: If minecraft crashes while connecting, you may need to restart into the most current version, set your render distance to 16 chunks or less, and then restart again into the mod version
note: You will probably want to increase the amount of memory allocated to minecraft: Launch Options >> Click on the Forge profile >> Change 'Xmx1G' to 'Xmx2G' or more if desired
